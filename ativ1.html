A Linguagem HTML

HTML (abreviacao para a expressao inglesa HyperText Markup Language, que significa Linguagem de Marcacao de Hipertexto) e uma linguagem de marcacao utilizada na construcao de paginas na Web. Documentos HTML podem ser interpreados por navegadores.

Ate este momentos, vimos os seguintes recursos da linguagem:

Recurso Tags associadas Descricao
Links <a> Criar links entre as paginas
Listas <ul> e <li> Criar listas de itens ordenados
<ol> e <il> Criar listas de itens nao ordenados
Imagens <img> Inserir imagens
